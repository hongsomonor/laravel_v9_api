- basic route : get and post
- create controller :
    -> php artisan make:controller RoleController --api  : use for build api
- create model :
    -> php artisan make:model Role -m >> use -m mean also create table
    -> table role {
        id
        name
        code
        description
        status
    }
    -> php artisan make:migration add_column_to_roles_table --table=roles >> use for add new column if we already migrate data
    -> php artisan migrate >> use for migrate all data into table
    -> php artisan migrate:reset >> use for reset all table in database
    -> php artisan migrate:refresh >> use for refresh all data in table on database
    -> php artisan migrate:status >> use for check status table

==> CRUD Category : (id , name , description , status , parent_id)
    -> php artisan make:model Category -m
        -m : migration file
    -> php artisan make:controller CategoryController --api
    -> Route::apiResource("Categories" , CategoryController::class);
        that apiResource use for CRUD api , auto have method of route
        no need to define route post get put delete
        just write one line do more
